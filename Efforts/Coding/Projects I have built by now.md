1. Job Scheduler
     A Genetic Algorithm Job Scheduler is a system that intelligently distributes tasks across multiple CPUs to minimize total execution time and balance workload efficiently.
     1. **Python** → Main programming language used for the whole project
     2. **Genetic Algorithm** → Core optimization technique inspired by natural evolution
     3. **Population Generation** → Created multiple possible scheduling solutions
     4. **Fitness Function** → Measured how efficient each schedule was
     5. **Selection** → Chose the best-performing schedules
     6. **Crossover** → Combined two good schedules to create better ones
     7. **Mutation** → Randomly modified schedules to explore new possibilities
     8. **Makespan Optimization** → Reduced total completion time across CPUs
     9. **Multi-CPU Scheduling** → Distributed jobs between multiple processors
